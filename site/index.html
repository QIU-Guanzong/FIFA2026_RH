方法论

为什么可解释：一条比分矩阵主干

铁律——比分矩阵是唯一真理源。所有市场（1X2 / 大小球 / 让球 / 波胆 / 出线 / 夺冠）都从同一个比分矩阵派生，绝不为单个盘口单独训黑箱模型，因此各盘口之间永远自洽。

01

国际赛 Elo 先验

1872 年至今全量国际赛，重要性加权 + 多趟暖启动，压低洲际通胀。

02

log‑linear λ

评分映射到每队进攻/防守强度，得出双方期望进球 λ、μ。

03

Dixon‑Coles 比分矩阵

双泊松 + 低比分相关性修正 τ，给出每个比分的概率。

04

统一派生盘口

对同一矩阵求和 → 1X2 / 大小球 / 让球 / 波胆，天然自洽。

05

赛会蒙特卡洛

官方分组 + 淘汰赛树联合模拟 4 万届，输出出线/夺冠率。

不止于预测：可验证的工程

每一项改进都用无泄漏回测或对真实结果的校准来量化，而非眼测。

多趟 Elo 评分

暖启动迭代把国际赛样本外 log loss 从 0.912 降到 0.897（无泄漏 walk‑forward 量化），压低洲际通胀。

ECE 0.04

自建 shot‑level xG

StatsBomb 开放数据 + 门口张角几何 + 可解释逻辑回归，对真实进球校准（非拟合 statsbomb_xg）。

OFC −70 Elo

洲际通胀校正

无泄漏增量 bootstrap 判定仅大洋洲（OFC）通胀稳健可校正，CAF/AFC 在噪声内 → 只校正前者，不一刀切。

85 项测试

纪律即代码

DC τ 符号、赛制恒等式（Σ出线=32 / Σ夺冠=1）、无泄漏守卫全部写进测试，每次改动自动校验。

诚实边界

这个模型还不能做什么

可解释的另一面是诚实标注不确定性与近似——以下都是已知边界，而非隐藏的缺陷。

  • 强弱来自我们的 Elo 先验，不是市场赔率——市场（锐盘）通常更准；本模型与之对比是健康的"略逊"，不是边际优势的证明。
  • 第三名落位取一组满足官方约束的合法匹配——495 种组合均非唯一，但对夺冠概率影响 ≈0.1pp（已量化锁定）。
  • 尚无阵容/首发层——天花板检验显示首发有真信号，但瓶颈是可解释的球员价值层（待建），非采购预测名单。
  • 东道主主场加成默认关闭——无法严谨 OOS 验证，仅作敏感性分析，不计入上面的数字。
  • 合成数据只验证机器正确性——任何"校准/准确度"结论都基于真实数据 + 严格 cutoff 回测。
